# 下一步优化方案

## 1. 优化目标

第一版已经完成“产品思路 + 数据结构 + 基线分析链路”的搭建。下一步优化要回答：

1. 是继续扩数据，还是先提模型精度。
2. 是继续围绕单事件深挖，还是扩展到多话题。
3. 是先强化研究能力，还是开始产品化展示。

## 2. 优化优先级建议

建议优先级如下：

1. 优先扩充真实样本数据
2. 再升级情感与观点识别模型
3. 最后再做更复杂的产品交互层

原因很明确：没有足够稳定的数据样本，模型优化的收益有限。

## 3. 数据层优化

### 3.1 扩大样本覆盖

1. 从单个事件扩展到多个事件
2. 从单一作者类型扩展到官方、媒体、自媒体、普通用户
3. 增加采样时间跨度，覆盖事件前期、爆发期、尾声期

### 3.2 提高字段完整度

1. 增加评论排序信息
2. 增加评论层级深度
3. 增加视频字幕/OCR 文本
4. 增加作者历史发文样本，用于动态画像

### 3.3 建立数据质量评估

1. 缺失率
2. 去重率
3. 时间格式一致性
4. 评论有效文本占比

## 4. 模型层优化

### 4.1 情感分析升级

从规则法升级为：

1. 规则法 + 轻量监督模型
2. 或规则法 + 大模型打标/校正

目标：

1. 提升反讽识别能力
2. 提升口语化表达识别能力
3. 提升多义句判断能力

### 4.2 观点抽取升级

当前是关键词驱动，后续可升级为：

1. 多标签分类
2. 主题聚类
3. 论点摘要

这样能更准确地区分：

1. 情绪表达
2. 事实补充
3. 责任追问
4. 谣言纠偏

### 4.3 传播角色识别升级

后续可引入更多特征：

1. 首次发声时机
2. 评论文本重复率
3. 跨视频活跃频次
4. 是否更偏向号召性扩散

注意：
这里应始终输出“疑似高活跃放大线索”，避免过度定性。

## 5. 分析目标优化

首版主场景是“突发事件舆情分析”。后续建议扩展两条支线：

1. 话题长期走势分析
   适合教育、就业、城市治理等议题。
2. 账号画像分析
   适合观察某类作者的叙事方式与受众反馈。

## 6. 流程优化

### 6.1 从单次分析变成标准流程

形成固定 SOP：

1. 选题
2. 样本采集
3. 清洗与脱敏
4. 运行分析
5. 复核结果
6. 输出汇报

### 6.2 引入人工校验闭环

对情感和观点结果抽样复核，建立误判样本池，反向优化规则和模型。

## 7. 产品化优化

如果后续继续做成系统，可加入：

1. 事件看板
2. 情绪趋势图
3. 观点簇分布图
4. 高活跃传播账号观察列表
5. 报告一键导出

## 8. 总体建议

对于当前阶段，最合理的下一步不是盲目追求更复杂技术，而是：

1. 用更多真实样本验证当前目标是否有价值
2. 找出最稳定、最常用的分析指标
3. 再决定模型和系统要往哪里加深

这会让项目从“能做”转向“值得做、可持续做”。
